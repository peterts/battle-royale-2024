# Nyetsartikkel til tegneseriestripe

Kode laget under Bouvet sin Battle Royale 2024. Løsningen bruker ChatGPT og Dall-E for å konvertere en nyhetsartikkel til en tegneseriestripe bestående av fire ruter.

## Eksempel

Nyhetsartikkel:
